Town of Shadows
===============

Quick start:
1. npm install
2. npm run dev
3. Build: npm run build
4. Deploy `dist/` to Netlify (drag-and-drop or connect GitHub repo)

This package is a playable prototype with text-only narration. It includes:
- 5x4 player grid
- Role pools per your configuration
- System narrator (text chat log)
- Private role modal (local)

Notes:
- This is a prototype. Advanced role interactions (full contact rules, detailed immunities, janitor clean, etc.) may require further development.
